# InsightCard Component

A reusable card component for displaying metrics and KPIs in the Method Platform UI.

## Installation

```bash
npm install @m-next/insight-card
```

## Usage

```jsx
import InsightCard from '@m-next/insight-card';

// Chart icon component
const ChartIcon = () => (
  <svg width="16" height="16" viewBox="0 0 16 16" fill="none">
    <rect x="2" y="10" width="3" height="4" fill="#545F67" />
    <rect x="6.5" y="6" width="3" height="8" fill="#545F67" />
    <rect x="11" y="2" width="3" height="12" fill="#545F67" />
  </svg>
);

function App() {
  return (
    <InsightCard
      title="Revenue distribution"
      value="60%"
      linkText="View customers"
      icon={<ChartIcon />}
      showInfoIcon
      infoTooltipContent="The percentage of your total revenue generated by your top customers."
    />
  );
}
```

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `title` | `string` | No | `''` | The title text displayed in the card header |
| `value` | `string \| number` | No | `''` | The metric value to display prominently |
| `linkText` | `string` | No | `null` | Text for the action link at the bottom |
| `onCardClick` | `function` | No | `null` | Callback function when the card is clicked (makes entire card clickable with hover effect) |
| `icon` | `ReactNode` | No | `null` | Custom icon component to display in the header |
| `showInfoIcon` | `boolean` | No | `false` | Whether to show the info icon in the header |
| `infoTooltipContent` | `string` | No | `null` | Tooltip content shown on hover of info icon |
| `isLoading` | `boolean` | No | `false` | Whether to show loading skeleton state |

## Examples

### Basic Card
```jsx
<InsightCard
  title="Total Sales"
  value="$125,000"
  linkText="View details"
/>
```

### With Custom Icon and Tooltip
```jsx
<InsightCard
  title="Active Users"
  value="1,234"
  icon={<CustomIcon />}
  showInfoIcon
  infoTooltipContent="The number of users who have been active in the last 30 days."
/>
```

### Loading State
```jsx
<InsightCard
  title="My pending estimates"
  icon={<ChartIcon />}
  isLoading
/>
```

### Clickable Card with Hover Effect
```jsx
<InsightCard
  title="Revenue distribution"
  value="60%"
  linkText="View customers →"
  icon={<ChartIcon />}
  showInfoIcon
  onCardClick={() => console.log('Card clicked!')}
/>
```

### Without Link
```jsx
<InsightCard
  title="Conversion Rate"
  value="23.5%"
  icon={<ChartIcon />}
/>
```

## Design

The component follows the Method Platform UI design system:
- Font: Source Sans Pro
- Border radius: 8px
- Border: 1px solid #BACAD0
- Background: #FFFFFF
- Padding: 16px
- Title color: #2A394A
- Value color: #2A394A
- Link color: #0D71C8
